Génère un JSON contenant les informations de 8 personnages, respectant les critères suivants :

Répartition des rôles :

3 personnages avec le rôle de 'suspect'.
3 personnages avec le rôle de 'temoin'.
1 personnage avec le rôle de 'victime'.
Caractéristiques pour chaque personnage :

Un champ nom contenant uniquement le nom de famille.
Un champ prenom contenant uniquement le prénom.
Un âge compris entre 18 et 85 ans.
Un sexe (homme ou femme).
Un rôle (suspect, temoin, victime ou assassin).
Contraintes : Renvoie uniquement un JSON correctement formaté en français. Le JSON doit respecter exactement les champs suivants pour chaque personnage : nom, prenom, age, sexe, role.
Ne mélange pas prénom et nom dans le même champ. Renvoie uniquement le JSON, sans texte explicatif.
